The Recordwhat was said · vs · what the documents showed

A searchable file of cases where an official account was contradicted by later evidence. Each entry pairs the statement at the time with what the record later showed, plus how it surfaced — and is tagged by how strong the proof is, so you can tell a settled case from a contested one at a glance.

Tier I — On the record
Admitted, declassified, or decided in court. A trail you can pull.
Tier II — Documented · contested
Strong documentary evidence; the official side disputes the scope or framing.
Tier III — Partial · unresolved
The cover story shifted, or a program was later admitted, but the full claim is unproven.

About the count. There are 181 cases here, not 1,500 — every one is a case the compiler could actually stand behind with a named trail, each verified against a primary source or major investigation before it went in. Click any file to open its dossier; direct links to the primary documents are being added file by file, each link verified before it ships. Padding it with entries that can't be checked would destroy the one thing that makes the archive worth keeping. To grow it honestly, draw from primary archives: the National Security Archive, the CIA FOIA reading room (CREST), GovernmentAttic, GovInfo, and agency reading rooms. Pull a document, verify it, then add it.

How to read a file. The most important field is How it surfaced — the mechanism that beat the cover-up: a named whistleblower, court discovery, a declassification act, records that survived a shredding order. Real suppressed truths crack along these lines and get sharper under scrutiny. The tier badge tells you how far that process has gone.

Add your own. Open this file in any editor, find the DATA array, copy a block. Fields: title, cat, tier (1–3), said, saidYear, shown, shownYear, how, source. Assign the tier honestly — Tier I only if there's an admission, declassification, or verdict. The rule that gives the archive its value: name the trail, or set it lower.
